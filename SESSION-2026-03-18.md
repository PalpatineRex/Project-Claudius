# Session 2026-03-18 — Voix opti finale

## Fixes appliqués cette session

### KinectVoice.py
- **CUDA cublas** : DLLs nvidia ajoutées au PATH avant ctranslate2 (cublas\bin + cudnn\bin)
  → faster-whisper tourne en GPU float16 (~0.3-1s par phrase)
- **Seuil fixe 600** : FIXED_THRESHOLD=600 remplace le seuil dynamique instable
- **Amara** : matching direct lowercase + apostrophes normalisées (sans unicodedata)
  HALLUCINATION_KEYWORDS liste simple, robuste sur encodages cassés
- **Anti double-envoi** : `_processing = threading.Lock()` + `acquire(blocking=False)`
  → utterances simultanées rejetées proprement
- **no_speech_threshold=0.6 + log_prob_threshold=-1.0** dans transcribe()
  → Whisper rejette lui-même les segments peu probables (hallucinations réduites)
- **crash `info.no_speech_prob`** : attribut inexistant dans cette version faster-whisper → supprimé

### KinectBridge.py
- **Mémoire conversationnelle** : _conversation_history[], MAX_HISTORY=6 échanges
  Haiku reçoit le contexte des 6 derniers tours → réponses cohérentes
- **Post-TTS lock** : time.sleep(0.5) après fin audio → évite auto-écho
- **TRANSCRIPT_FILE** : réponses Claudius écrites dans transcript.txt

### KinectTranscript.py (nouveau)

- Dark UI, David en bleu, Claudius en vert
