PIPELINE VOIX (ton architecture):
- Micro: Bird UM1 (device 1, configurable via CLAUDIUS_MIC_DEVICE)
- Reconnaissance: faster-whisper small, CUDA float16 GPU, seuil VAD adaptatif max(1000, ambiant*1.5), filtre hallucinations (no_speech_threshold=0.4, logprob_threshold=-0.5, minimum 2 mots)
- LLM: Claude Haiku via API Anthropic (cle dans api_key.txt > env var ANTHROPIC_API_KEY, strip guillemets auto)
- TTS: Voix blend Jessica+SIWIS 50/50 (Piper CUDA, 2 modeles boot ~4s, synth parallele 2 threads, blend spectral v3c: DTW enveloppe + warp interp + phase Jessica + magnitudes mixees + gate silence + HF preserve consonnes, ~60ms blend, ~1.1s total)
- Audio: sounddevice (cross-platform, synthese RAM, pas de WAV disque), detection auto mute quand video/musique joue (pycaw)
- Gestes: KinectMotor.exe (oui/non/blink/hello/think/reset), auto-blink
- Chemins: portables via env vars CLAUDIUS_* + relatifs au script + sys.executable
- CUDA: auto-detecte via site.getsitepackages() (cublas, cudnn, cufft, cuda_runtime, cusolver, cusparse, nvjitlink)
- System prompt: cache par mtime du fichier contexte
- Logs: rotation 500 lignes
